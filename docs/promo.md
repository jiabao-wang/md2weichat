# 我写了个工具，让 Markdown 一键发布微信公众号

> 如果你也在用 Markdown 写作，却在发布公众号时被排版折磨，这篇文章就是为你写的。

## 那个困扰我很久的问题

作为一个习惯用 Markdown 写技术文章的人，每次发布公众号都是一场噩梦：

- 😩 **代码块粘贴后格式全乱**，缩进丢失、高亮消失，手机端显示一坨
- 😩 **排版样式无法复用**，每篇文章都要手动调字号、间距、颜色
- 😩 **图片一张张手动上传**，一篇文章十几张图点到手酸
- 😩 **手机端预览和电脑不一样**，列表多出奇怪的圆点，代码被截断

我试过 MDNice、Online-Markdown 等在线编辑器，但它们要么需要联网粘贴、要么主题要付费、要么批量处理很麻烦。作为一个开发者，我想要的是：**一个能在本地运行、支持命令行、所见即所得、一键搞定的工具**。

于是我花了几个晚上，写了一个开源工具——**md2wechat**。

## md2wechat 是什么？

一句话介绍：**把本地 Markdown 文件一键转成公众号排版 HTML，并发布到草稿箱**。

它解决的核心问题：

1. 🎨 **11 套精美主题**，从经典配色到金融时报、纽约时报、GitHub、赛博朋克风格，一键切换
2. 💻 **代码块手机端完美换行**，这是我踩了无数坑才解决的硬骨头
3. 🖼️ **本地图片自动上传**到微信素材库，不用手动一张张传
4. 🖥️ **可视化 Web 界面**，在线编辑、实时预览、所见即所得
5. 📦 **批量发布**，选目录或多选文件，进度条展示，失败可单独重试
6. 📋 **命令行工具**，支持脚本和 CI/CD 集成
7. 🔒 **完全本地运行**，AppID/AppSecret 只存在你自己电脑上

## 它长什么样？

启动后是一个简洁的 Web 界面，左边是文章信息和文件操作，右边是编辑器和预览：

- **✏️ 编辑标签**：在左栏填写标题、作者、摘要，选择主题和封面，中间区域直接写 Markdown
- **👁️ 预览标签**：实时渲染成公众号样式，和最终效果一致
- **📑 分屏标签**：左边写 Markdown 右边实时看效果，写技术文章利器
- **📦 批量发布**：选一个目录，所有 Markdown 文件自动解析，每篇可以单独设置封面，一键批量上传到草稿箱

界面使用微信绿色主题，干净清爽，没有多余的东西打扰写作。

## 代码块手机端的坑，我帮你踩完了

这是我花时间最多的地方，也是最想分享的技术细节。

**问题**：微信公众号编辑器不支持 `<style>` 标签，所有样式必须内联；微信移动端 WebView 对 CSS 支持有限，`overflow-x:auto` 横向滚动在手机上经常失效，导致代码被截断或换行混乱。

**我尝试过的方案**：

- ❌ `white-space: pre` + `overflow-x:auto` → 手机端不支持横向滚动，代码被截断
- ❌ `<span style="display:block">` 包裹每行 → 部分安卓微信内核对 inline 转 block 渲染不稳定
- ❌ `word-break: break-all` → 太激进！会在 `/`、`→`、`.` 等符号处随意断行，比如 `Vercel / Netlify` 被断成三行
- ❌ 依赖外部 CSS class → 微信编辑器会把 class 全部剥离

**最终方案**：

每一行代码在 HTML 结构层面就被包裹为独立的 `<section>` 块级元素，每行都自带完整的内联样式，**不依赖任何 CSS 也能正确换行**。空行也保留高度，缩进完整。词的断开策略使用 `word-break: break-word`，只在真正超长的连续字符串（比如长 URL）处断行，不会把正常代码拆得七零八落。

这是经过大量真机测试验证的方案，效果稳定。

## 5 分钟快速上手

### 第一步：安装

```bash
git clone https://github.com/jiabao-wang/md2weichat.git
cd md2wechat
npm install
npm run build
```

### 第二步：配置微信凭证

1. 登录微信公众平台 → 开发 → 基本配置
2. 获取 AppID 和 AppSecret
3. 把你的 IP 加到白名单
4. 运行配置命令：

```bash
npm link   # 可选，全局使用 md2wechat 命令
md2wechat config 你的AppID 你的AppSecret
```

### 第三步：启动界面

```bash
md2wechat web
```

浏览器会自动打开 http://localhost:3000，开始写作！

### 或者用命令行

```bash
# 预览不发布
md2wechat convert my-article.md --preview

# 一键发布草稿
md2wechat convert my-article.md --draft --cover ./cover.jpg -t github
```

## 批量发布：系列文章的救星

如果你有一批历史文章要迁移到公众号，批量发布功能会是你的福音：

1. 切换到"📦 批量发布"标签
2. 点"📂 选择目录"，选一个文件夹，所有 Markdown 文件自动识别
3. 设置默认作者和主题，每篇可以单独改标题、换封面
4. 点"🚀 开始批量发布"，看进度条逐个完成
5. 失败的文章点"🔄 重试"单独重发，不用从头来
